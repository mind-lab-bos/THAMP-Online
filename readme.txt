Readme

Thamp1banalysis.m is outdated and for process previous versions of the online participant data.

thamp1banalysis_nomusic.m is the newer version for processing the participant data from psytoolkit that includes the no music condition.

thamp1banalysis_nomusic_batch.m is the newest version and will be the final code for this part of the project. This code will analyze all 400 final participants and also complete time sensitive performance analyses.

thamp1b_liking_familiarity.m is used to compare subjects liking and familiarity with their performance and with modded and unmoved conditions. This will need to be updated to run with the final batch of participants.
